# PHIGO — Hemsidor som ger dig fler kunder

PHIGOs egen hemsida. Bygger moderna, konverteringsoptimerade hemsidor åt lokala småföretag.

---

## Erbjudande

| Del | Pris |
|---|---|
| Hemsida | 8 000 kr (engång) |
| Drift | 399 kr/mån |
| Leveranstid | 14 dagar |

---

## Tech Stack

- **Framework:** Next.js (App Router)
- **Språk:** TypeScript
- **Styling:** Tailwind CSS 4 + shadcn/ui
- **Animationer:** Framer Motion
- **Formulär:** React Hook Form + Zod
- **Pakethanterare:** pnpm

---

## Kom igång

```bash
pnpm install
pnpm dev
```

Öppna [http://localhost:3000](http://localhost:3000).

## Bygg för produktion

```bash
pnpm build
pnpm start
```
